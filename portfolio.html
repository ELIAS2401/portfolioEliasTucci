Experiencia en NP Construcciones

Desarrollé el sitio web oficial de NP Construcciones, una empresa constructora que antes se manejaba únicamente por recomendaciones pero necesitaba dar el salto al entorno digital. El proyecto incluyó la creación de una página moderna, clara y totalmente adaptable, con información de la empresa, sus servicios, proyectos realizados y medios de contacto. Además, gestioné la compra del dominio npconstrucciones.com.ar en DonWeb, configuré correos corporativos y diseñé flyers y tarjetas con QR para que el cliente pueda compartir su web de forma profesional.

  • ✔️ Reunión con el cliente para entender necesidades reales y definir estructura del sitio.
  • ✔️ Desarrollo completo del sitio institucional (HTML, CSS, Bootstrap, JS).
  • ✔️ Integración de secciones: Inicio, Nosotros, Servicios, Proyectos y Contacto.
  • ✔️ Compra y configuración del dominio en DonWeb + setup de DNS.
  • ✔️ Configuración de correos corporativos (info@npconstrucciones.com.ar).
  • ✔️ Creación de flyer y tarjeta con QR para promoción del sitio.
  • ✔️ Diseño responsive adaptable a móviles, tablets y escritorio.
  • ✔️ Asesoramiento al cliente sobre presencia digital y buenas prácticas.

Experiencia en DeizerDistribuidora

Desarrollé un sitio web completo para una distribuidora de ferretería que antes mostraba su catálogo únicamente mediante PDFs. El objetivo del proyecto fue crear una plataforma dinámica, fácil de actualizar y sin costos de hosting backend, para que el cliente pudiera manejar su catálogo sin depender de un programador. Para lograrlo, transformé cientos de productos en un Excel estructurado (nombre, precios, unidades por bulto, descuentos, categorías, imágenes y descripciones). Luego generé un JSON consumido por JavaScript, desde el cual la web construye automáticamente el catálogo, los filtros por categoría y el detalle de cada producto. Además, implementé un carrito de compras funcional: suma precios según tipo de producto, muestra totales en tiempo real, permite vaciar el carrito y genera un enlace de WhatsApp con el pedido listo para enviar. Finalmente, enseñé al cliente a actualizar su catálogo modificando el Excel, convirtiéndolo a JSON y subiéndolo a GitHub, asegurando que pudiera mantener el sitio por su cuenta.

  • ✔️ Tratar con un cliente real, entender sus necesidades y traducirlas en soluciones técnicas.
  • ✔️ Organizar un catálogo grande y convertirlo en un formato manejable (Excel → JSON).
  • ✔️ Crear interfaces dinámicas con JavaScript puro, sin backend
  • ✔️ Diseñar un sistema de productos con categorías, filtros y buscador.
  • ✔️ Implementar un carrito de compras con: sumatoria automática, actualización en tiempo real, envío del pedido por WhatsApp, uso de localStorage para mantener los items.
  • ✔️ Optimizar un sitio sin servidor, usando GitHub Pages y archivos estáticos.
  • ✔️ Capacitar al cliente para mantener el contenido, dándole un flujo simple de actualización.
  • ✔️ Manejo de Bootstrap, responsive design y pequeñas utilidades UX (scroll to top, modal, etc).
  • ✔️ Paciencia y gestión de cambios constantes pedidos por el cliente.
  • ✔️ Experiencia real de trabajo: reuniones, feedback, idas y vueltas, organización del tiempo y entregables.

Experiencia en Rocio Del Cielo

Desarrollé un sitio web liviano y visual para un emprendimiento de pastelería casera. Incluye catálogo de productos, galería de imágenes optimizadas, sistema de pedidos integrado por WhatsApp, diseño responsive y una estética cálida enfocada en transmitir calidad artesanal. El objetivo fue ofrecer una experiencia simple, clara y enfocada en la conversión (pedidos rápidos).

  • Aprendí a:
  • ✔️ A trabajar con clientes reales, interpretando necesidades simples pero importantes (estética, colores cálidos, claridad del mensaje).
  • ✔️ A seleccionar, editar y optimizar imágenes de productos para que la web se vea profesional sin perder carga rápida
  • ✔️ A crear formularios de pedido simples usando links preformateados a WhatsApp.
  • ✔️ A estructurar una web completa con HTML + CSS + Bootstrap, priorizando diseño limpio y accesible.
  • ✔️ A implementar un pequeño catálogo dinámico usando JSON y JavaScript para cargar productos.
  • ✔️ A diseñar una experiencia orientada a móviles, ya que la mayoría de clientes compra desde el teléfono.
  • ✔️ Capacitar al cliente para mantener el contenido, dándole un flujo simple de actualización.
  • ✔️ Manejo de Bootstrap, responsive design y pequeñas utilidades UX (scroll to top, modal, etc).
  • ✔️ Paciencia y gestión de cambios constantes pedidos por el cliente.
  • ✔️ Experiencia real de trabajo: reuniones, feedback, idas y vueltas, organización del tiempo y entregables.
